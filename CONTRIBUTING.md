# المساهمة في المشروع (Contributing) 🤝

نرحب بجميع المساهمين لتحسين وتطوير منصة محاكاة المقابلات بالذكاء الاصطناعي! إذا كنت ترغب في المساهمة، يرجى اتباع الإرشادات التالية:

## خطوات المساهمة 🛠️

1. **النسخ (Fork):** قم بعمل Fork للمستودع إلى حسابك الخاص.
2. **إنشاء فرع (Branch):** قم بإنشاء فرع جديد للميزة أو الإصلاح الذي ستعمل عليه:
   ```bash
   git checkout -b feature/AmazingFeature
   ```
3. **البرمجة (Code):** قم بإجراء تغييراتك والتأكد من عملها بشكل سليم.
4. **الاعتماد (Commit):** قم بعمل Commit لتغييراتك برسالة واضحة:
   ```bash
   git commit -m 'Add some AmazingFeature'
   ```
5. **الرفع (Push):** ارفع تغييراتك إلى مستودعك:
   ```bash
   git push origin feature/AmazingFeature
   ```
6. **طلب سحب (Pull Request):** افتح Pull Request جديد في المستودع الأصلي واشرح فيه التغييرات التي قمت بها.

## قواعد ومعايير كتابة الكود 📜
- **TypeScript:** المشروع يعتمد على TypeScript، لذا تأكد من تحديد الأنواع (Types) وعدم استخدام `any` إلا عند الضرورة القصوى.
- **التصميم:** استخدم **Tailwind CSS** مع المتغيرات المعرفة مسبقاً للحفاظ على هوية بصرية موحدة (Aesthetics).
- **الترجمة (i18n):** لا تقم بإضافة نصوص ثابتة (Hardcoded) في واجهات المستخدم. استخدم دائماً دالة الترجمة `t()` وأضف النصوص في ملفات `src/i18n/ar.json` و `en.json`.
- **الأداء:** استخدم `React.memo` و `useMemo` عند الحاجة لمنع إعادة التصيير غير الضرورية.

شكراً لمساهمتك معنا في بناء هذه المنصة! 🚀
